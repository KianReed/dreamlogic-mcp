# 🌌 dreamlogic-mcp
专业的周公解梦 MCP 服务端 | 让 AI 读懂古老的梦。支持全量结构化古籍数据检索。

Developed by **KianReed**

> **让 AI 读懂古老的梦。**
> A professional Model Context Protocol (MCP) server for ancient Chinese dream interpretation.

## 🌟 项目亮点
- **全量数据**: 内置 3000+ 条深度校对的《周公解梦》结构化数据。
- **工业级标准**: 完全遵循 MCP 协议规范，支持多块内容返回（UI 表格 + 结构化 JSON）。
- **智能匹配**: 自动清洗梦境描述，支持核心意象提取与模糊检索。
- **零依赖配置**: 快速集成至 Claude Desktop, Cursor, 或任何支持 MCP 的环境。
